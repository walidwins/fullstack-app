# TP8 - Full Stack App (React + Node.js + MongoDB)

## 📌 Description

Application full-stack réalisée dans le cadre du TP8 de Développement Web Avancé.

Le projet contient :

- Un frontend développé avec React.js
- Un backend développé avec Node.js et Express.js
- Une base de données MongoDB Atlas
- Une authentification sécurisée avec JWT

---

## ⚙️ Technologies utilisées

### Frontend
- React.js
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs
- Helmet
- express-rate-limit
- dotenv
- cors
